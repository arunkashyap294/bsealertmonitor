2. Restart the program (Required if you ever update 

config.yaml
 to add more companies):

bash
sudo systemctl restart bse-monitor
3. Stop it temporarily:

bash
sudo systemctl stop bse-monitor
You can now safely close your SSH browser window. The VM will keep doing its job and you'll get Telegram alerts on your phone whenever there's a new announcement! Let me know if you want to add any other features in the future.

If you update the config.yaml file, you must restart the process for the changes to take effect. The background service reads the configuration file when it first starts up and keeps that in memory; it doesn't automatically detect changes to the file while running.

Whenever you add or remove a company in config.yaml, just run this single command on your VM:

bash
sudo systemctl restart bse-monitor
This will instantly stop the current monitor and start a new one, reading your fresh config file in the process.
